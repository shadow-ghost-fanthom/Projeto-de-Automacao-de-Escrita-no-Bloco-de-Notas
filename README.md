# 🤖 Automação de Escrita com PyAutoGUI
Este é um script simples em Python que automatiza a escrita de linhas de texto no Bloco de Notas. Você monta uma lista com o que quer escrever e o código faz o trabalho de digitação por você utilizando teclas de atalho.

⚠️ Nota: Este é o meu primeiro projeto utilizando a biblioteca pyautogui, desenvolvido para entender os conceitos básicos de automação de interface!

⌨️ Atalhos do programa:
Tecla Windows: Abre o Bloco de Notas.

Tecla F2: Clica na tela e digita toda a lista criada.

Tecla Esc: Minimiza as janelas e encerra o script.

📌 Aviso: As coordenadas do clique (pyautogui.click(x=242, y=149)) foram configuradas especificamente para a resolução da minha tela. Se for testar, pode ser necessário ajustar esses valores.
